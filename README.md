## Overview

This workshop provides a hands-on implementation of a **multi-agent AI system** using the [Microsoft Agent Framework](https://learn.microsoft.com/en-us/agent-framework/) and [Azure AI Foundry](https://azure.microsoft.com/en-us/products/ai-studio/). The sample application demonstrates podcast guest research and outreach email composition through coordinated AI agents.
